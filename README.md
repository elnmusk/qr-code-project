### What I learned

I learned and revised the use of HTML/CSS and I'm looking forward to work on other projects. I learned that I can center stuff using grid which is what I used to center my box.

```css
body {
    height: 100vh;
    margin: 0;
    display: grid;
    place-items: center;
}
```
